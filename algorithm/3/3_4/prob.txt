正整数 n に対して，原点に中心を置く半径 n の円盤内（周上の点も含む）にある整数格子点の個数を C(n) と書くことにする。 n →∞ のとき C(n) / n2 は円周率 π に近づいていくはずある。この話に基づいて π の値の近似値を求めることを考える。

入力として n を与えると， C(n) を数え上げて C(n) / n2 を double型で計算して出力するプログラムを作れ。C(n) は long型で数え上げよ。

ちなみに，π の値はつぎのとおりである。
3.14159 26535 89793 23846 26433 83279 ...
【入力】
正整数 n　１つ。

【出力】
 n と C(n) / n2 とを，書式を "%d:%.15f\n" として書き出せ。

【例】
入力例
1000
出力例
1000:3.141549000000000
