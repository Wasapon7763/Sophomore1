非負整数 n ( n<263) の整数平方根 x（非負整数 x:  (x)2 ≦ n ＜ (x+1)2 ) を計算する関数 long lsqrt(long n) を作りたい。作った関数が正しく動作するかどうかを確認する検査用の main 関数をもった，つぎの仕様をもつプログラムとして仕上げよ。
　・標準入力として整数 n を与える
　・ n < 0 なら，negative argument  というメッセージを１行として出力する。
　・ n ≧ 0 なら，整数平方根を計算し，その値を１行として出力する。
なお出力の末尾には改行を書き出すこと。
